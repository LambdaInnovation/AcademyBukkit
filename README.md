AcademyBukkit
=============
A Bukkit plugin allowing [AcademyCraft][ac] to support some Bukkit plugins

Runtime Dependencies
--------------------
+ Required:
    * AcademyCraft
+ Supporting:
    * PlotMe
    * Residence

Building Dependencies
---------------------
+ Bukkit

[ac]:  https://github.com/LambdaInnovation/AcademyCraft
